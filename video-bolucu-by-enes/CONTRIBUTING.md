# Katkı Rehberi

Video Bölücü by Enes açık kaynak bir projedir. Katkılar memnuniyetle incelenir, ancak projeye dahil edilecek her değişiklik maintainer onayından geçmelidir.

## Katkı Akışı

1. Repoyu fork edin.
2. Değişikliklerinizi ayrı bir branch üzerinde yapın.
3. Gerekiyorsa Android Studio veya Gradle ile test edin.
4. Pull request açın.
5. Açıklamada neyi değiştirdiğinizi ve nasıl test ettiğinizi belirtin.

## Kabul Kriterleri

- Kullanıcı verisi toplanmamalı veya üçüncü taraflara gönderilmemelidir.
- İnternet, reklam veya analitik SDK'sı eklenmemelidir.
- İmza anahtarı, parola, yerel path, AAB/APK çıktısı veya özel dosya commit edilmemelidir.
- Türkçe kullanıcı arayüzü metinleri bozulmamalıdır.
- Değişiklik mevcut basit kullanım akışını karmaşıklaştırmamalıdır.

## Güvenlik

Güvenlik açığı veya yanlışlıkla eklenmiş gizli bilgi fark ederseniz bunu public issue olarak paylaşmadan önce maintainer ile özel kanaldan iletişime geçin.
