# Güvenlik Politikası

Bu repoya keystore, parola dosyası, AAB/APK çıktısı, release arşivi veya kişisel yerel yapılandırma dosyası eklenmemelidir.

Özellikle aşağıdaki dosyalar gizli kabul edilir:

- `keystore.properties`
- `*.jks`
- `*.keystore`
- `*.p12`
- `*.pem`
- `*.aab`
- `*.apk`
- `*.zip`
- `local.properties`
- `release-final/`
- `release-packages/`

Bir güvenlik sorunu fark ederseniz public issue açmadan önce maintainer ile özel olarak iletişime geçin.
